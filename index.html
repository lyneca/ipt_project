About Sense

Sense is a project that puts large scale enterprise data collection in your hands. Demos include temperature, light, sound, pressure and humidity data, but with the Sense API, the potential for environmental data collection is limitless.

To those genuinely interested in the product, I apologise.

In reality, this is a school based software assignment involving the marketing of a product. While this product does work, it is probably not ready for legitimate use due to the specificity of the hardware involved.

Idle Mode

WiFi indicators

The demo board features LEDs on the WiFi module to indicate connection status - blue flashes every time a signal is sent, red shows power to the module.


Transmit Mode

Status LEDs

Four powerful LEDs on the bottom left of the board indicate state progression - red represents waiting for a request, orange represents collecting data, green represents power, and blue represents data transmission.
